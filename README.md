## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/Saki-Video-Conferencing-Platform.git
cd Saki-Video-Conferencing-Platform
```

### 2️⃣ Install dependencies

```bash
# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
```

### 3️⃣ Run the application

```bash
# Start backend
cd server
npm start

# Start frontend
cd ../client
npm start
```
